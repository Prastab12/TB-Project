# Tuberculosis Data Standardization & Harmonization Project

## Overview
This project processes raw IIHMS TB Data, cleans and harmonizes it, maps it to WHO TB indicators, standardizes it into FHIR Measure/MeasureReport format, and provides gap analysis and analytical dashboards.

## Architecture
1. Data Cleaning & Harmonization Layer
2. Indicator Operationalization Layer
3. FHIR Standardization Layer
4. FHIR Validation Layer
5. Gap Analysis Layer
6. Analytics & Visualization Layer
