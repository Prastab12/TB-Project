SECTION A: PHASE 5 OVERVIEW
================================================================================

Phase 5 is the capstone phase — it publishes all prior work as a formal,
standards-compliant HL7 FHIR R4 Implementation Guide (IG).

The IG specifies:
  - How TB surveillance variables are represented as FHIR Measure resources
  - How monthly aggregate data is stored as FHIR MeasureReport resources
  - How Nepal's Bikram Sambat (BS) calendar is encoded alongside Gregorian dates
  - How under-reported months are flagged using data-absent-reason: not-reported
  - Canonical shared instances for NTP and Kathmandu District

SECTION F: IG PROFILES
================================================================================

  TBMeasure (tb-measure):
    - Covers 32 variables (26 ratio + 6 cohort)
    - Enforces scoring from MeasureScoring ValueSet (ratio | cohort)
    - Requires url, identifier, version, name, title, status, date, publisher,
      description, group, population, criteria

  TBMeasureReport (tb-measure-report):
    - 1,920 instances: 32 variables x 60 months
    - status fixed = #complete; type fixed = #summary
    - subject constrained to Location; reporter to Organization
    - period carries NepaliCalendarPeriod extension (bs-year, bs-month, fiscal-year)
    - measureScore system fixed to UCUM; code fixed to #1
