     * Numerator and Denominator Counts are both absent.

     * Denominator Count is absent.

3. Structural JSON Representation:
   - Counts: Numerical count fields are omitted, and standard FHIR primitive extensions

     ```json
     "_count": {
       "extension": [{
         "url": "http://hl7.org/fhir/StructureDefinition/data-absent-reason",
         "valueCode": "not-reported"
       }]

     ```

     ```json

       "unit": "ratio",
       "system": "http://unitsofmeasure.org",
       "_value": {
         "extension": [{
           "url": "http://hl7.org/fhir/StructureDefinition/data-absent-reason",
           "valueCode": "not-reported"
         }]
       }
     }
     ```
